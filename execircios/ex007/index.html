Principais formas de Formatação

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B

Nesta frase temos um termo em destaque usando a tag STRONG (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântico)

Nesta frase temos um termo em ênfase usando a tag EM (semântico)

Texto marcado

Nesta frase temos um termo marcado

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto deletado

Podemos marcar um texto como excluído usando a tag DEL

Texto inserido

Podemos marcar um texto sublinhado com a tag U (não semântico)

E podemos também sublinhar um texto de inserção com a tag INS

Texto Sobrescrito

Para inserir coisas do tipo ( x20 + 3 ) use a tag SUP

Texto subscrito

Para escrever coisas do tipo ( H2O ) use a tag SUB
